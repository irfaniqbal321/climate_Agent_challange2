# Climate Agent Challenge 2 - Sehat Saathi

## Project Overview
Sehat Saathi ek AI-powered health assistant hai jo climate change ke effects ko monitor karta hai aur health advice deta hai. Ye app Google Antigravity ke zariye vibe coding se banai gai hai.

## How Google Antigravity Was Used
- Antigravity se agent_backend code generate kiya
- Task planning aur implementation Antigravity ke zariye ki
- Debugging aur code optimization Antigravity ne ki

## Architecture
- **agent_backend/**: AI agents jo health data process karte hain
- **sehat_saathi/**: Frontend mobile app
- **Tech Stack**: Node.js, JavaScript, AI Agents

## How to Run
1. `git clone https://github.com/irfaniqbal321/climate_Agent_challange2.git`
2. `cd sehat_saathi` ya `cd agent_backend`
3. `npm install`
4. `node index.js`
